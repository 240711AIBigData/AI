파이썬
---
- 프로그래밍 언어
- 다른 언어에 비해 쉬움(동적자료형 형태)
- 다양한 라이브러리(패키지) → 빅데이터, 인공지능
- 오픈소스
- 기본 확장자는 .py 이자만, 노트북 확장자는 .ipynb
```
  .py : 일반적인 파일, 결과를 보려면 매번 실행해야 함

  .ipynb : 실행 결과를 바로 볼 수 있고, 그 결과도 함께 저장됨
    - 실행 방법
      - Ctrl + Enter : 현재 셸 실행
      - Shift + Enter : 현재 셀 실행 후 다음 새로운 셸 생성
```
  
</details>
