# 데이터 전처리(data preprocessing)
- 분석에 적합하게 데이터를 가공하는 작업
- pandas : 전처리 작업에 가장 많이 사용되는 패키지

|함수|기능|
|:-:|-|
|query()|행 추출|
|df[]|열(변수) 추출|
|sort_values()|정렬|
|groupby()|집단별로 나누기|
|assign()|변수 추가|
|agg()|통계치 구하기|
|merge()|데이터 합치기(열)|
|concat()|데이터 합치기(행)|
