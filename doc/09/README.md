# 데이터 전처리(data preprocessing)
- 분석에 적합하게 데이터를 가공하는 작업
- pandas : 전처리 작업에 가장 많이 사용되는 패키지

|함수|기능|
|:-:|-|
|query()|행 추출|
|df[]|열(변수) 추출|
|sort_values()|정렬|
|groupby()|집단별로 나누기|
|assign()|변수 추가|
|agg()|통계치 구하기|
|merge()|데이터 합치기(열)|
|concat()|데이터 합치기(행)|

<br>

파이썬에서 사용하는 기호
---
|논리 연산자|기능|     |산술 연산자|기능|
|:-:|-|-|:-:|-|
|<|작다|   |+|더하기|
|<=|작거나 같다|   |-|빼기|
|>|크다|   |*|곱하기|
|>=|크거나 같다|   |**|제곱|
|==|같다|   |/|나누기|
|!=|같지 않다|   |//|나눗셈의 몫|
|\||또는(or)|   |%|나눗셈의 나머지|
|&|그리고(and)|   |   |   |
|in|매칭 확인|   |   |   |
