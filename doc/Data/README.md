# 실습 데이터 파일
실습 데이터 파일. 다운로드해서 워킹 디렉터리에 삽입한 다음 실습 진행.

## 다운로드 방법
- ① 다운로드할 파일 클릭
- ② `Raw` 버튼 또는 `Download` 버튼 마우스 오른쪽 클릭
- ③ [다른 이름으로 링크 저장] 클릭


## CSV 파일 다운로드 방법
- ① [파일 이름' 뒤 확장자 `txt`를 `csv`로 변경 
- ② 파일 형식'을 '모든 파일'로 변경 → [저장]
